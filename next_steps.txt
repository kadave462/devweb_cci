Vous allez implémenter des formulaires pour ajouter des équipes et des matchs.

Les étapes que vous allez devoir réaliser sont les suivantes :

Ajouter des routes pour générer les formulaires d'ajout d'équipes et de matchs ;
Valider les données des formulaires en ajoutant des contraintes de validation ;
Afficher les erreurs de validation dans les formulaires ;
Ajouter des listes déroulantes pour sélectionner les équipes dans le formulaire d'ajout d'un match ;
Utiliser le service pour ajouter les équipes et les matchs dans la base de données.
Vous allez devoir utiliser les concepts et technologies suivantes vus en cours :

Utilisation de Thymeleaf pour générer des formulaires et afficher les erreurs de validation ;
Utilisation des annotations pour valider les données des formulaires.
